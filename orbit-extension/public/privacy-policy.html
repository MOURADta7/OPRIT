Privacy Policy for ORBIT

Last updated: February 15, 2026

TL;DR: ORBIT is designed with privacy as its foundation. We collect ZERO data. Everything stays on your device. Your API keys are encrypted. We literally cannot see anything you do.

1. Our Privacy Promise

ORBIT is built on a simple principle: Your data is yours, and it stays yours.

Unlike most software, ORBIT:

  • ✅ Does not collect any user data
  • ✅ Does not track your usage
  • ✅ Does not use analytics or telemetry
  • ✅ Does not store data on our servers (we don't have any)
  • ✅ Does not sell or share any information
  • ✅ Does not use cookies

2. What Data We Store

All data is stored locally on your device only:

API Keys

  • Your OpenAI, Claude, Gemini, and Groq API keys
  • Encrypted using AES-256-GCM before storage
  • Encrypted with device-specific keys derived from your browser
  • Keys are only decrypted when making API calls
  • Keys are never transmitted to ORBIT servers (we don't have any)

Usage Data

  • Your API usage statistics (tokens used, costs incurred)
  • Stored in your browser's local storage only
  • Automatically deleted after 90 days
  • Used only for budget tracking and cost optimization

Settings

  • Your preferences (budget limits, display settings, etc.)
  • Stored locally in browser storage

3. What We Do NOT Store

ORBIT does not access, store, or process:

  • ❌ Your sales data or revenue information
  • ❌ Customer comments or personal information
  • ❌ Your marketplace dashboard content
  • ❌ Any business metrics or analytics
  • ❌ Your browsing history
  • ❌ Any data from your sales platforms

4. Third-Party Services

ORBIT connects directly to AI providers that you choose to enable:

OpenAI

  • Only if you provide an API key
  • Data sent: Your prompts and context for AI generation
  • See OpenAI's Privacy Policy: openai.com/privacy

Anthropic (Claude)

  • Only if you provide an API key
  • Data sent: Your prompts and context for AI generation
  • See Anthropic's Privacy Policy: anthropic.com/privacy

Google (Gemini)

  • Only if you provide an API key
  • Data sent: Your prompts and context for AI generation
  • See Google's Privacy Policy: policies.google.com/privacy

Groq

  • Only if you provide an API key
  • Data sent: Your prompts and context for AI generation
  • See Groq's Privacy Policy: groq.com/privacy

Important: Your API keys are sent ONLY to the respective AI provider's API endpoints. They are never sent to ORBIT or any intermediary servers.

5. Data Security

We take security seriously:

  • Encryption: API keys encrypted with AES-256-GCM using device-specific keys
  • Local Storage: All data stored in chrome.storage.local (isolated from websites)
  • No Server: ORBIT has zero backend servers - everything is client-side
  • Open Source: Code is open for security audit (coming soon)
  • Minimal Permissions: Only requests permissions absolutely necessary

6. Your Rights

You have complete control over your data:

  • ✅ Access: All your data is visible in the extension settings
  • ✅ Export: You can export all your data anytime from settings
  • ✅ Delete: You can delete all data from settings with one click
  • ✅ Uninstall: Uninstalling removes ALL data automatically
  • ✅ No Account: No account required, no data to "delete"

7. Children's Privacy

ORBIT is not intended for use by children under 13. We do not knowingly collect personal information from children under 13.

8. Changes to This Policy

We may update this Privacy Policy from time to time. Changes will be posted here with a new "Last updated" date. Since ORBIT doesn't collect your contact information, we can't notify you directly - please check this page periodically.

9. Compliance

ORBIT is designed to comply with:

  • ✅ GDPR (General Data Protection Regulation)
  • ✅ CCPA (California Consumer Privacy Act)
  • ✅ Chrome Web Store Developer Program Policies

Because we collect zero data, compliance is straightforward!

10. Technical Details

For the technically curious:

  • Storage: chrome.storage.local API (browser extension storage)
  • Large Data: IndexedDB (client-side database)
  • Encryption: Web Crypto API (PBKDF2 + AES-256-GCM)
  • Network: Direct fetch() calls to AI provider APIs only
  • No external libraries for data collection or analytics
